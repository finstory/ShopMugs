# ShopMugs
First you need install Node.js... 
<br/>
https://nodejs.org/en/download
<br/>
<br/>
Open Api (You must inside to path ./api) :
<br/>
1- "npm install"
<br/>
2- "npm start"
<br/>
<br/>
Open Client (You must inside to path ./client) :
<br/>
1- "npm install --global yarn"
<br/>
2- "yarn dev"
